## 操作方法

### フリック操作（メイン）
画面を指でスワイプしてパックマンを操作します。
- 上にスワイプ → 上移動
- 下にスワイプ → 下移動
- 左にスワイプ → 左移動
- 右にスワイプ → 右移動

指を離さずにドラッグし続けることで、連続的に方向転換が可能です。

### ボタン操作
- BtnB（中央）: ゲーム開始 / 一時停止
- BtnA（左）: リセット
